Welcome to IMSN Shared Resources Guide

This guide will help new users navigate GitHub for accessing and sharing files across unconnected hospitals within the Irish Medications Safety Network (IMSN).

Section 1: Structure of the Repository

The main repository is named IMSN_Shared_Resources and is organized as follows:

  • Project Type
    • Categories like protocols, guidelines, or data collection tools.
  • Contributor
    • Folders named after contributors sharing resources.
  • Version Folders
    • Each version of the file is stored in its own folder, labeled with version numbers.

Section 2: Naming Conventions

To keep files organized and easily identifiable, please use the following file-naming format:

[ProjectType]_[Contributor]_[YYYYMMDD]_[FileDescription]_v[VersionNumber].ext

Example:

DRC_ICU_Colleague3_20241113_DataCollection_v1.0.csv

  • ProjectType: Short code for the project (e.g., DRC_ICU for Drug Range Checking in ICU).
  • Contributor: Your name or identifier (e.g., Colleague3).
  • YYYYMMDD: Date in YYYYMMDD format (e.g., 20241113 for November 13, 2024).
  • FileDescription: Brief description of the file.
  • v[VersionNumber]: Version number starting with v1.0.
  • .ext: Appropriate file extension (e.g., .csv, .docx).
Tip: Consistent and descriptive naming helps everyone quickly identify and locate files, enhancing collaboration and reducing errors.

Section 3: How to Use GitHub

To access the repository:

  • Navigate to the IMSN_Shared_Resources repository on GitHub.
  • Browse through the folders categorized by project type and contributor.
  • Click on a file to view it, and click the "Download" button to save it to your computer.
Note: Access to certain files may be restricted. Please ensure you have the necessary permissions.

Section 4: Version Control

GitHub tracks file versions to show updates over time. Look for a version label (e.g., v1.0, v1.1) to identify the latest file version.

When updating a file, increment the version number accordingly and update the VersionNumber in the filename.

The actual repository itself can be located at IMSN Shared Resources
